本次审计针对新增的注册/搜索功能及 SQLite 数据库进行安全加固，修复了 SQL 注入、密码存储、认证同步等 10 项安全问题。

📋 涉及功能范围

  • 新增用户注册功能（含 SQLite 数据库存储）
  • 新增用户搜索功能（按用户名/邮箱搜索）
  • 登录认证从字典迁移到 SQLite 数据库
  • 输入校验与频率限制机制
