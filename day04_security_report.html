🔍 文件内容深度验证

+ import magic
+ from PIL import Image
+ from io import BytesIO

+ def validate_file_content(file_stream, filename):
+     # 1. 检查文件大小（限制5MB）
+     file_stream.seek(0, 2)
+     file_size = file_stream.tell()
+     if file_size > MAX_FILE_SIZE:
+         return None, "文件大小超过5MB限制"

+     # 2. 检查真实MIME类型（防止扩展名伪装）
+     content = file_stream.read()
+     actual_mime = magic.from_buffer(content, mime=True)
+     if actual_mime not in ALLOWED_MIME_TYPES:
+         return None, "不支持的文件类型"

+     # 3. 验证扩展名与MIME类型匹配
+     ext = filename.rsplit('.', 1)[1].lower()
+     if ext not in ALLOWED_MIME_TYPES[actual_mime]:
+         return None, "文件扩展名与内容不匹配"

+     # 4. 重新处理图片，去除恶意代码
+     img = Image.open(BytesIO(content))
+     img.verify()  # 验证是否为有效图片
+     img = Image.open(BytesIO(content))
+     output = BytesIO()
+     img.save(output, format=img.format, quality=85)
+     # 重新保存，去除嵌入的PHP/JS等恶意代码

+     return output, None

💡 安全建议

🔒 额外安全措施建议：

1. 文件上传目录权限：设置uploads目录为不可执行（chmod -x）
2. 定期扫描：使用杀毒软件定期扫描上传目录
3. 文件类型限制：根据业务需求进一步限制文件类型和大小
4. CDN/OSS存储：考虑将上传文件存储到独立的文件服务器或云存储
5. 日志审计：记录所有文件上传操作，便于追踪和审计
6. WAF防护：部署Web应用防火墙提供额外保护层
