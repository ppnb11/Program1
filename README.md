# 用户管理系统

一个基于 Flask 的简易用户信息管理平台，支持登录、登出、用户信息展示功能。

## 功能

- 🔐 用户登录（含 CSRF 保护）
- 👤 用户信息展示
- 🚪 登出功能

```bash
cd /opt/Class01

```
